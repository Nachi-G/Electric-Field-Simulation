# Electric Field Visualization

This is an interactive 3D simulation of the electric field from a Charge Ring/Disk, emphasizing symmetry and vector cancellation.

## How to Run

1.  **Install Python**: Ensure you have Python installed.
2.  **Install Dependencies**:
    Open a terminal/command prompt in this folder and run:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the Simulation**:
    ```bash
    python electric_field_viz.py
    ```

## Controls
-   **Left Click + Drag**: Orbit/Rotate Camera
-   **Right Click + Drag**: Pan Camera
-   **Scroll**: Zoom
-   **Sliders**: Adjust Position, Radius, Charge, and Count (N).
-   **Toggles** (Top-Right): Show/Hide vectors and switch between Ring/Disk modes.
